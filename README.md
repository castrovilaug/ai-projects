# AI Engineer Portfolio

Proyectos de integración de IA para empresas. Construidos con Groq (Llama 3.3 70B), LangChain, Streamlit y Python.

## Proyectos

### 1. Chat con historial — Semana 1
Chat conversacional con memoria de contexto usando Llama 3.3 70B vía Groq API.

**Stack:** Python · Groq API · Llama 3.3 70B  
**Caso de uso:** Base para cualquier asistente de empresa con memoria de conversación.

---

### 2. Lector de PDFs con IA — Semana 2
Sube cualquier PDF y hazle preguntas en lenguaje natural. La IA responde basándose únicamente en el contenido del documento.

**Stack:** Python · Groq API · Llama 3.3 70B · PyPDF2 · Streamlit  
**Caso de uso:** Consulta rápida de contratos, manuales, catálogos o documentación interna.  
**Demo:** [Ver aplicación en vivo](https://ai-projects-9kbf2jdprwjwbdtnbqkrrl.streamlit.app/)
